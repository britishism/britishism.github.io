## Versioning
The release versioning system is `yy.mm.dd` which:
* `yy` stands for "Year",
* `mm` stands for "Month",
* And `dd` stands for "day".
 
For example: release `16.09.22` is the release of September 22<sup>nd</sup>, 2016.
